# Product Images for Furnessence

## Image Requirements

Place your product images in this folder with the following specifications:

### File Naming Convention:
- product-1.jpg
- product-2.jpg
- product-3.jpg
... and so on up to product-19.jpg

### Image Specifications:
- **Format**: JPG, PNG, or WebP
- **Recommended Size**: 800x800px (square format works best)
- **Max File Size**: 2MB per image
- **Aspect Ratio**: 1:1 (square) preferred
- **Quality**: High resolution, clear product shots

## Notes:
- If you don't have product images yet, the site will still work (images will appear as broken links)
- You can use placeholder images from sites like:
  - https://placeholder.com/
  - https://unsplash.com/ (free furniture images)
  - https://pexels.com/ (free stock photos)
  
## Adding New Products:
1. Place the image in this folder
2. Note the filename
3. Add the product through the Admin Panel
4. Enter the image path as: `./assets/images/your-image-name.jpg`

## Example Paths:
```
./assets/images/product-1.jpg
./assets/images/product-2.jpg
./assets/images/my-custom-chair.jpg
```

Remember: The path in the database should start with `./assets/images/`
